## Prepare project for use:

1- sail artisan migrate </br>
2- sail artisan db:seed </br>

To give the admin role to the admin user we run the following command: </br>
sail artisan shield:super-admin --user=2

User Credentials:
- Admin (id-2)
    - Email: admin@admin.com
    - Password: password
- User (id-3)
    - Email: user@user.com
    - Password: password
